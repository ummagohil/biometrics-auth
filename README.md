# 🔑Biometric Auth App

Authenticate users with biometric authentication and Supabase.

### 📋Aims of App

- Authenticate users with biometric authentication
- Store user data in Supabase

### 👩‍💻Technical Details

- React
- Next.js
- Shadcn UI
- TailwindCSS
- TypeScript
- Supabase

### 🔧How to Run the App

```bash
  npm run dev
```

### 💭Future Improvements

- Unit tests
- React 19 compatibility with Supabase
